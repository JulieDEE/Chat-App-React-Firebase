### Installation

1. Get a free Firebase account : <a href="https://firebase.google.com/">ðŸ”¥</a>

2. Clone the repo
   ```sh
   git clone [https://github.com/JulieDEE/Chat-App-React-Firebase.git]
   ```

   ```sh
   yarn install
   ```
